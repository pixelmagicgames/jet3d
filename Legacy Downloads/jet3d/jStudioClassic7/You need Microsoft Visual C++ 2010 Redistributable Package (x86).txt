You need Microsoft Visual C++ 2010 Redistributable Package (x86) (MSVCR100.dll) installed on your system for jStudioClassic apps to work on your computer.

Go to the following URL and get the redistributable, install it on your machine.

http://www.microsoft.com/downloads/en/details.aspx?familyid=A7B7A05E-6DE6-4D3A-A423-37BF0912DB84&displaylang=en

jStudioClassic will run on Windows XP (SP3), Windows 7, and probably other XP or later versions with the latest service packs installed.

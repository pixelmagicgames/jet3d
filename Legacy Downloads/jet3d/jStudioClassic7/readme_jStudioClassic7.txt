You may download the runtime library installer at the following link:
http://www.microsoft.com/downloads/en/details.aspx?FamilyID=a7b7a05e-6de6-4d3a-a423-37bf0912db84

Mission of the jStudioClassic series 
The mission of the jStudioClassic series is to maintain and restore legacy Jet3D applications with the main purpose of keeping it operational on the latest Windows versions.  
This series focuses on legacy functionality present up through 2008. The mission is not interested in advancing the feature set of Jet3D. The main development branch of Jet3D is involved in adding new features such as the addition of scripting functions, new file formats and the Direct3D9 driver.
Coders involved in the jStudioClassic series emphasize code updating and improved code design for increased stability and robustness to the existing/legacy codebase.

tStudio3d7 was built in 2011 using Microsoft Visual Studio 2010.

In order to run tStudio3D7 applications, your system must have the Microsoft Visual C++ 2010 runtime dll installed.

You may download the runtime library installer at the following link:
http://www.microsoft.com/downloads/en/details.aspx?FamilyID=a7b7a05e-6de6-4d3a-a423-37bf0912db84

If this link does not work, do a google search with the following string:
microsoft visual c++ 2010 redistributable package
